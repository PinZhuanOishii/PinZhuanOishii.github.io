一般吃法

烤烏魚子可以單獨食用，也可以搭配其他料理，如蘋果、大蒜苗。它帶有豐富的鮮味和淡淡的鹹香，質地細膩，口感豐富。每片烤烏魚子散發出的濃郁風味令人難以抗拒，讓人欲罷不能。

材料:
  • 烏魚子
  • 白蘿蔔
  • 大蒜苗
  • 蘋果
  • 水梨
步驟:
  1. 準備烤盤和烤箱：先預熱烤箱至攝氏200度（華氏400度），同時在烤盤上鋪上烤盤紙或使用不沾烤盤。
  2. 清洗烏魚子：將烏魚子從包裝中取出，放入碗中，用冷開水輕輕漂洗幾次，去除多餘的鹽分和雜質。
  3. 烹調：若要烹調烏魚子，切記不要過度烹調，如此便能吃到烏魚子的鮮甜美味。
  4. 烤製：將烏魚子均勻地分佈在預熱好的烤盤上，確保烤盤上的烏魚子呈單層分散排列。將烤盤放入預熱好的烤箱中，如果吃要較彈牙的口感，則以中火烤約3分鐘；如果吃要較綿密不黏牙的口感，則以中火烤約3到5分鐘。
  5. 休息：取出烤好的烏魚子，讓它稍微放涼，以便風味更好地散發出來。烤熟的烏魚子表面應該有輕微的脆感，但內部仍然保持柔軟的口感。

烏魚子春捲

烏魚子春捲融合了烏魚子的豐富鮮味和春捲的口感，帶來獨特的風味體驗。這道菜可以作為前菜、點心或派對小吃，給食客帶來驚喜和滿足感。記得根據自己的口味調整調味料的份量，並隨喜好添加其他蔬菜或配料。

材料：
  • 春捲皮
  • 烏魚子
  • 蔬菜（例如生菜、紅蘿蔔、青瓜等）
  • 紅蘿蔔絲和大蒜絲（調味用）
步驟:
  1. 準備食材：將蔬菜洗淨並切絲，烏魚子從包裝中取出備用。
  2. 食材炒熟：在熱鍋中加入少許油，放入紅蘿蔔絲和大蒜絲，快速翻炒至蔬菜稍軟，保持蔬菜的鮮嫩口感。然後加入烏魚子翻炒約一分鐘，使其均勻受熱。
  3. 包裹春捲：將春捲皮放在工作台上，放上適量的蔬菜和烏魚子混合物，並將春捲皮捲起來，注意將兩側收緊。可以在捲的過程中塗些水在春捲皮的邊緣，以幫助黏合。
  4. 煎春捲：將平底鍋中加入足夠的油，燒熱後放入春捲，用中火煎炸。煎至春捲皮呈金黃色和脆軟口感，並將春捲翻面繼續煎炸。
  5. 取出並瀝乾：將煎好的烏魚子春捲取出，放在廚房紙巾上瀝乾多餘的油份。
  6. 切片和享用：將春捲切成適當大小的塊狀，擺盤後即完成一道美味料理。

烏魚子炒飯

烏魚子炒飯結合了烏魚子的鮮味和炒飯的香氣，呈現出一道豐富多樣的料理。您可以根據個人喜好加入其他蔬菜、肉類或海鮮，創造出屬於自己的獨特口味。享受這道美味的烏魚子炒飯！

材料:
  • 白飯
  • 烏魚子
  • 蔬菜（例如胡蘿蔔、豌豆、玉米等）
  • 香菇（可選）
  • 雞蛋
  • 蔥
  • 洋蔥
  • 蒜頭
  • 醬油
  • 鹽和胡椒粉
  • 植物油
步驟:
  1. 準備食材：將蔬菜和香菇洗淨，切成小塊；洋蔥和大蒜剁碎；將白飯事先煮好，冷卻至室溫。
  2. 炒蔬菜和香菇：在同一個鍋中加入更多的植物油，放入洋蔥和大蒜，炒至香味散發。接著加入蔬菜和香菇，繼續翻炒直到蔬菜熟軟。
  3. 加入白飯和調味料：將冷卻好的白飯加入鍋中，用鍋鏟拌勻，使白飯和蔬菜充分混合。加入醬油，適量的鹽和胡椒粉，根據個人口味調整調味料的份量。
  4. 炒蛋液：將雞蛋打入碗中，打散成蛋液。在鍋的一側移開飯菜，倒入蛋液，輕輕攪拌，讓蛋液均勻地包裹在飯菜上。
  5. 翻炒均勻：將蛋液與飯菜一起翻炒，確保飯菜和蛋液均勻混合，直到炒飯均勻加熱。
  6. 加入烏魚子：將烏魚子盡可能的切碎，在起鍋前加入到炒飯內，並立即拌勻。
  7. 完成：當炒飯拌勻後，即可將烏魚子炒飯盛盤，搭配其他配菜享用。
